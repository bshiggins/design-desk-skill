# Explore Context

Output mode when comparing options or picking design direction.

## Style
- Show 2-3 concrete options with visual descriptions
- Compare trade-offs for each option
- Lead with your recommendation and reasoning
- Include creative arsenal suggestions when the motion dial supports it

## Behavior
- When possible, generate a playground HTML file with the options side-by-side
- Let the user toggle/compare visually before deciding
- Don't implement until the user picks a direction
- Ask one question at a time, not a wall of options

## Verbosity
Medium. Enough to make an informed choice, not so much it's overwhelming.
